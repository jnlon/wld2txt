# wld2txt

`wld2txt` is a program that converts a [Terraria](https://terraria.org/) World
file (.wld) into ASCII text.

# Usage

`wld2txt` may either accept the contents of a wld file through stdin, or a path
to a wld file as its first argument. All output is printed to stdout, so you
will want to use shell redirection to capture it to a file.

### Example

`./wld2txt ~/.local/share/Terraria/Worlds/myworld.wld > myworld.txt`

You can then view `myworld.txt` in a text editor and scroll around, though make
sure you disable text wrapping since each line is at least a few thousand
characters long.

# Building

You need the following software to build it:
  * ocaml compiler (ocamlopt)
  * make

Once installed, `cd` into the root directory and run 'make'. 

An executable called `wld2txt` will be generated.
